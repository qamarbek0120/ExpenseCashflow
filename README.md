# CashFlow Expense Manager App

## Overview

The CashFlow Expense Manager App is a web application developed to help users manage their expenses. It provides features to add, view, and categorize expenses.

## Features

- **Expense List:** Displays a list of all expenses.
- **Expense Detail:** Shows detailed information about a specific expense.
- **Create Expense:** Allows users to add a new expense.

## Technologies Used

- **Backend:** ASP.NET Core, Repository Pattern
- **Frontend:** Angular

## Setup

1. Clone the repository.
2. Install dependencies for both the backend and frontend (`npm install` for Angular).
3. Configure the backend to connect to your database.
4. Start the backend (`dotnet run`) and frontend (`ng serve`) servers.

## Usage

1. Access the application at `http://localhost:4200`.
2. Use the Expense List to view all expenses.
3. Click on an expense to view its details in Expense Detail.
4. Use Create Expense to add a new expense.
